What is Lithcoin?

LTCO is a utility token built on Polygon for fast, low-cost transactions. It powers payments within the McJohnson Wallet and enables seamless transactions across a growing ecosystem of mini-apps.

Utility Token

LTCO is designed for transactional use—not as an investment or security. It facilitates payments and settlements within applications.

Polygon Network

Built on Polygon PoS for sub-second confirmations and transaction costs under $0.01.

Growing Ecosystem

LTCO is integrated with the McJohnson Wallet and will support mini-apps built using our SDK, creating a community-driven utility network. App Coming Soon

How to Buy LTCO

Three ways to acquire Lithcoin utility tokens.

Option 1: McJohnson Wallet Coming Soon

  1. Download the McJohnson Wallet ('App')( will be available on iOS and Android)
  2. Create or import your wallet
  3. Use the built-in LTCO integration to acquire tokens directly
  4. Access mini-apps and use LTCO for payments within the ecosystem

Note: The McJohnson Wallet is a non-custodial wallet. You control your private keys at all times.

Explore more about the McJohnson Wallet here

Option 2: QuickSwap (Decentralized Exchange)

  1. Install MetaMask and add the Polygon network
  2. Get POL (Polygon's native token) for gas fees
  3. Visit QuickSwap
  4. Connect your wallet and search for LTCO using the contract address
  5. Swap USDC for LTCO
